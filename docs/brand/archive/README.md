# Archived Design Material

This folder preserves old Cold Start brand directions, Paper explorations, generated mockups, and retired review notes.

Archived files are historical context only. They are not current app guidance, not implementation specs, and not the default source for image prompts. Use the root `DESIGN.md` for the live app visual system.

If an archived direction becomes relevant again, first compare it against the rendered app and update `DESIGN.md` deliberately. Do not silently import old typography, dark-shell layouts, or Paper mockup details back into current work.
